**Design notes:**
- Primary command: `virsh domstats --raw` or `virsh list --all --uuid` for domain enumeration
- Secondary command (optional): `virsh domblklist <domain>` for snapshot inventory
- Graceful degradation: if virsh missing or command fails, return default snapshot with available=false
- Use `if let Ok(...)` for non-critical secondary commands (snapshot age may not be available on all systems)

  - **Step 3 (Optional):** Use `if let Ok(o) = run_timed("virsh", &["snapshots", ...], profile)` to enrich snapshot info
    - Optional means collector doesn't fail if this command is missing (non-critical data)
